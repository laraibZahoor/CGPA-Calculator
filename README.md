CGPA Calculator (C++ Console App)
Description
This program calculates CGPA based on:
Grades (A, B, C, D, F)
Credit hours for each course
It uses weighted average method.

Features
Multiple courses input
Grade to grade-point conversion
Credit hours support
Accurate CGPA calculation

Grade Point Table
A = 4.0
B = 3.0
C = 2.0
D = 1.0
F = 0.0

Formula
CGPA = (Sum of (Grade Point × Credit Hours)) / (Total Credit Hours)

Example

Enter number of courses: 3

Course 1:
Grade: A
Credit Hours: 2

Course 2:
Grade: B
Credit Hours: 2

CGPA = 3
